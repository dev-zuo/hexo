小程序web-view组件内嵌网页调用微信JS-SDK

发表于 2018-02-24 | 分类于 微信小程序 |

小程序内嵌网页调用JSSDK与公众号网页里面调用基本一致，需要注意的地方：

  • 内嵌网页调用JS-SDK需要使用公众号的appid及appsecret，与小程序的appid没有关系
  • 小程序内嵌网页只支持部分JSSDK，详情参见： 小程序web-view组件
